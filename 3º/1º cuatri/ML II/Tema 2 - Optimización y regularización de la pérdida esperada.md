## Función de pérdida
Medimos cómo de malo es el modelo para un único dato. Se ve así:$$L(y_i, f(x_i;w))$$Siendo $f(x_i;w)$ igual a $y_i$. Un ejemplo podría ser el error cuadrático:$$L(y_i, \hat{y_i})=(y_i-\hat{y_i})²$$
