## Resumir presentaciones
De cara a resumir presentaciones, lo cual hago mucho, uso esta, funciona bien con Gemini 3.1 Pro o ChatGPT 4-5

```
Eres un tutor experto en la materia de los apuntes adjuntos. Tu tarea es generar un resumen que cumpla estas tres condiciones simultáneamente:

  

COMPLETO: No puede faltar ningún concepto, dato, definición, fórmula, proceso, matiz o caso especial presente en el original. Revisa el documento entero antes de escribir.

  

SINTETIZADO: Elimina redundancias y relleno. Agrupa ideas relacionadas. Usa el mínimo de palabras que garantice máxima fidelidad informativa.

  

COMPRENSIBLE: Conserva las explicaciones del original. Si alguna no es suficientemente clara, amplíala con una explicación propia marcada como (Explicación adicional: ...). Añade ejemplos propios si ayudan, marcados como (Ejemplo: ...).

  

Formato: Usa encabezados jerarquizados, negrita para términos clave, listas numeradas para procesos ordenados y viñetas para el resto. Termina con una sección "Conexiones clave" con 5-8 frases sobre las relaciones más importantes entre conceptos.

  

Antes de responder, verifica: ¿está cada concepto del original en el resumen? ¿Hay explicaciones que necesiten ampliarse? ¿Es más corto que el original sin perder información? Solo envía el resumen cuando las tres respuestas sean sí.
```

## Generación de infografías con NotebookLM
Para tener esquemas conceptuales que ayuden a afianzar un concepto, uso este prompt para afinar la herramienta de infografía del producto [NotebookLM](https://notebooklm.google/) 

```
Ensure generous empty space between all sections and elements, leave 30-40% of the design as breathing room to boost comprehension by 20% and prevent visual overwhelm.
```
